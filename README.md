# APCalculadora


## Descrição
Calculadora Simples, feita como projeto para a Academia do Programador, pode realizar operações simples e  tabuadas.

## Funcionalidade

- Soma
- Subtração
- Multiplicação
- Divisão
- Tabuada*
- Histórico** 

*O programa pode realizar a tabuada do 1 ao 10 de qualquer número

**O programa salva todas as operações feitas em um histórico que pode ser acessado pelo menu principal
